# 🛒 E-Commerce Solution

A scalable and modern E-Commerce Solution built to simulate real-world online shopping workflows. The project demonstrates best practices in software architecture, backend development, authentication, product management, order processing, and business logic implementation.

---

## 📖 Overview

This project was created as a practical learning experience to explore how modern e-commerce platforms are designed and implemented.

The solution focuses on delivering a complete shopping experience, including product browsing, cart management, user authentication, order handling, and administrative operations.

---

## ✨ Features

### 👤 Authentication & Authorization

* User registration and login
* Secure authentication
* Role-based authorization
* Protected endpoints

### 📦 Product Management

* Create, update, and delete products
* Product categorization
* Product inventory management

### 🛒 Shopping Cart

* Add products to cart
* Update quantities

### 👨‍💼 Administration

## 🚀 Getting Started

### Prerequisites

Make sure you have installed:

* .NET SDK
* SQL Server
* Visual Studio or VS Code

### Clone the Repository

```bash
git clone https://github.com/SaifEldeen11/E-Commerce_Solution.git

cd E-Commerce_Solution
```

### Configure Database

Update your connection string in:

```json
appsettings.json
```

### Apply Migrations

```bash
dotnet ef database update
```

### Run the Application

```bash
dotnet run
```
